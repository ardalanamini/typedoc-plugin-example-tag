## Prerequisites

`npm i -D typedoc`

## Installation

`npm i -D typedoc-plugin-example-tag`

## Usage

`typedoc ... --plugin typedoc-plugin-example-tag --preferred-example-language javascript ...`

> The `preferred-example-language` option is set to `typescript` by default
